# Sistema de Gestión Hotelera

Proyecto académico desarrollado como parte de la carrera de Ingeniería en Sistemas Informáticos.

## Descripción
Sistema orientado a la gestión básica de habitaciones y reservas de un hotel, permitiendo organizar la información de disponibilidad y registro de clientes, con enfoque en el control interno de datos.

## Tecnologías utilizadas
- C#  
- MySQL  

## Funcionalidades principales
- Registro y gestión de habitaciones  
- Control de disponibilidad  
- Registro de clientes  
- Gestión básica de reservas  

## Alcance del proyecto
Proyecto de carácter académico enfocado en la aplicación de conceptos de desarrollo de sistemas de información y bases de datos relacionales. No está orientado a uso comercial.

## Aprendizajes
- Modelado de bases de datos relacionales  
- Desarrollo de aplicaciones en C#  
- Conexión y manejo de bases de datos MySQL  
- Aplicación de lógica de negocio  
- Organización de información para sistemas administrativos  

## Nota
Proyecto desarrollado con fines académicos.
